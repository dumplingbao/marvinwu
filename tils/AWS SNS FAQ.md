* how to grant sls right to invoke sns ?

Ref: https://hackernoon.com/a-crash-course-on-serverless-with-aws-triggering-lambda-with-sns-messaging-bc17d9f81ca2

```yml
# serverless.yaml

provider:
  name: aws
  runtime: python3.6
  stage: ${env:STAGE, 'test'}
  timeout: 600
  region: ${env:AWSREGION}  
  iamRoleStatements:
    - Effect: "Allow"
      Resource: "*"
      Action:
        - "sns:*"


```
